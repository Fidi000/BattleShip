How to Play

Go to the Login Page, found here: login

If you have already created an account, type in your username and password into the boxes.
If you don't have an account, just type in a username and password into the boxes.

If you already have an account, just hit the blue 'Login' button and you will be taken to the game board.

If you are creating a new account, just hit the red 'Sign Up' button and your account will automatically be created.
If someone already has the username you entered, try a different username.
